My tmux settings explained

I am using tmux for almost 2 years now. I was using and was a big fan of screen before that. But screen was replaced by tmux from the moment one of my colleague introduced me to a more actively developed tmux. Here I am documenting my Tmux configuration. Full configuration file is stored at Github #Use screen key combination unbind C-b set -g prefix C-a bind C-a send-prefix Above settings will replace default prefix for tmux from Ctrl+b or CMD+b to Ctrl+a or CMD+a.
