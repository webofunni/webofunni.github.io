Centralized login using PAM

Pluggable authentication module

Provides API for application to handle authentication, session management, accounting etc

example PAM conf : reboot

root@host [~]# cat /etc/pam.d/reboot 
#%PAM-1.0
auth       sufficient   pam_rootok.so
auth       required     pam_console.so
#auth       include     system-auth
account    required     pam_permit.so

Chrooting services and users

Chrooting services and users provides isolation and security
